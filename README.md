# CS7641-group-19

This is the Github repo for group 19's CS7641 project (Spring 2025).


# CODE FILES:

**1. Enformer Python Notebook -**
   Implementing Enformer model to get regulatory sequence information for genes we are interested in. 
   
**2. Preprocessing file (using R) -**
   Extracted Ovarian Cancer data from TCGA and filtered top 500 highly variable genes. Extracted metadata from these samples and genes. Saved everything to seperate CSV files. 
   
**3. KNN Implementation Python Notebook -**
     Implemented KNN supervised learning for Ovarian Cancer gene expression data using tumor classification as the target variable. Observed 62% accuracy with a 70-30 train-test split.
